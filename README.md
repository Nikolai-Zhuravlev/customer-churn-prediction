# Прогнозирование снижения покупательской активности клиентов

## Постановка задачи
Интернет-магазин «В один клик» заметил снижение активности постоянных клиентов. Цель проекта — построить модель, которая предскажет вероятность снижения покупательской активности клиента в ближайшие 3 месяца, и выделить сегменты покупателей для персонализированных предложений.

## Используемые модели
- **K-Nearest Neighbors (KNN)** — подбор числа соседей
- **Decision Tree** — подбор глубины дерева
- **Logistic Regression** — подбор параметра регуляризации C
- **Support Vector Classifier (SVC)** — подбор параметра C

## Результаты
| Модель | ROC-AUC (валидация) |
|--------|---------------------|
| SVC (OneHot+Std) | **0.9116** |
| LogReg | 0.90+ |
| KNN | 0.85+ |
| Tree | 0.80+ |

**Лучшая модель:** SVC с ROC-AUC = 0.9068 на тестовой выборке.
